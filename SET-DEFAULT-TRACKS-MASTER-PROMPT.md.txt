## 3. HARD REQUIREMENTS ‚Äî byte-exact safety (NON-NEGOTIABLE)

These are the rules that keep the feature from corrupting a user's only copy of a
file. Every one must hold in code and be proven by a test.

- **BYTE-EXACT GUARANTEE.** After an edit, every byte is identical to before
¬Ý EXCEPT (a) the intended flag byte(s), and (b) in the Void-reuse case, the
¬Ý inserted flag element + the shrunk `EbmlVoid` header/payload it consumed.
- **WRITABILITY GATE = `Os.fstat(fd)` ‚Üí require `S_ISREG` + a seekable fd**
¬Ý (lseek SEEK_END size matches the document size). **NEVER probe by writing a
¬Ý test byte** ‚Äî a test write corrupts providers with replace-semantics. A
¬Ý pipe/socket/char-device fd (typical of cloud/virtual DocumentsProviders) ‚Üí
¬Ý **FAIL-SAFE SKIP**(reason="provider not writable").
- **DURABILITY** via `Os.fdatasync` (size never changes on the in-place paths).
¬Ý Treat fsync/fdatasync as durable ONLY for local / SD / USB regular files; SKIP
¬Ý anywhere it is unreliable. SAF rename is **not** atomic ‚Äî never call it atomic;
¬Ý keep the original until the rewritten copy re-probes clean.
- **EBML PARSER IS THE SINGLE SOURCE OF TRUTH** for both analysis and the write
¬Ý target. Locate the TrackEntry by the byte offset captured during analysis and
¬Ý **re-validate its TrackNumber at write time** (TOCTOU). FFprobe/MediaInformation
¬Ý may be used ONLY as an optional cross-check oracle, NEVER to drive a byte write
¬Ý (ffprobe stream index ‚âÝ Matroska TrackNumber). Duplicate/absent TrackNumber ‚Üí
¬Ý SKIP that file.
- **ALL-OR-NOTHING PER FILE via a pre-image journal**: before the first data
¬Ý write, stash the ORIGINAL bytes of EVERY region you will modify + a content
¬Ý signature covering the **actual target region** (not just size+first/last 64
¬Ý bytes). fsync the journal BEFORE writing. A crash/cancel rolls back ALL
¬Ý regions. On restore, verify size + region bytes; on any mismatch DO NOT restore
¬Ý ‚Äî surface needs-manual-review.
- **SINGLE CONTIGUOUS WRITE** for the Void-reuse path: buffer the whole span
¬Ý `[firstChanged, lastChanged]` (including untouched bytes between), one
¬Ý `Os.write`, one fdatasync. Recompute ALL offsets from the FINAL post-edit
¬Ý layout; never mix pre-edit and post-edit offsets.
- **VERIFICATION after every write, before DONE**: (a) semantic re-probe asserts
¬Ý the INVARIANT ‚Äî exactly the chosen audio resolves default=1 and all other audio
¬Ý resolve 0; ‚â§1 subtitle default as chosen; forced as chosen; (b) structural
¬Ý byte-diff asserts ONLY the exact allowed offsets changed (assert the
¬Ý changed-region count equals expected) + file length unchanged (in-place);
¬Ý Void-reuse also re-parses end-to-end with every Cluster header at its ORIGINAL
¬Ý offset. **(c) A byte-diff failure OVERRIDES a passing semantic check ‚Üí FAILED ‚Üí
¬Ý roll back.** Prefer an independent oracle (mkvinfo/ffprobe) for the one-default
¬Ý check ‚Äî a masked-SHA from the engine's own allowed-set is circular.
- **NEVER write a flag whose desired value already equals its implied default
¬Ý when the element is absent** (already a no-op). Never mass-mutate flags the
¬Ý user did not choose (e.g. do not clear other subtitles' FlagForced).

## 4. The EBML edit engine ‚Äî decision table (full detail in the design doc)

Per (track, flag, desired value); implied default = FlagDefault‚Üí1, FlagForced‚Üí0.

| # | Element present? | Desired vs current | Action |
|---|---|---|---|
| 1 | Yes, `dataSize==1` | flip value | **Size-preserving 1-byte write** at `element.endOffset-1`; fdatasync |
| 2 | Yes | already desired | **No-op** (success) |
| 3 | No | == implied default | **No-op** (absence already means this) |
| 4 | No | ‚âÝ implied default (the COMMON "clear default on an absent-FlagDefault track" case) | **INSERT via Void-reuse** (Path B) |
| 5 | No, case 4, **no usable Void** | ‚Äî | **Tail-rewrite** (Path C ‚Äî DEFERRED; Phase A = FAIL-SAFE SKIP) |
| 6 | any | risk gate trips | **FAIL-SAFE SKIP**(reason) |

**Always FAIL-SAFE SKIP (with reason, never silent):** `dataSize==0` wanting
non-zero; `dataSize>1` (multi-byte flag); unknown-size Tracks/TrackEntry;
multi-Segment; Tracks after the first Cluster or beyond the read window;
duplicate/absent TrackNumber; DocType ‚àâ {matroska, webm}; fd not regular/seekable;
provider rejects `"rw"`; a position VINT won't fit its original width; content
signature mismatch (file changed underneath).

**Void-reuse (Path B):** donor `EbmlVoid` restricted to **inside the target
TrackEntry or directly inside Tracks ONLY** (never elsewhere in the Segment ‚Äî
that moves intervening bytes). `need = insLen + Œ£(size-VINT width growth on the
LCA path)`; require `oldVoidTotal ‚àí need ‚â• 2`; **never change the Void's
size-VINT width**. Derive `insLen` from the actual encoded INS bytes
(FlagDefault=0 = `88 81 00`; FlagForced=1 = `55 AA 81 01`). Plan ALL inserts for a
file against one mutable layout in a single pass; if the Void(s) cannot cover all
required inserts, **SKIP the whole file** (preserve the "exactly one default"
invariant ‚Äî never apply a partial subset).

**Tail-rewrite (Path C ‚Äî spec it, mark DEFERRED):** must also fix the **Segment
size VINT**, every SeekHead `SeekPosition` whose target ‚â• Tracks (enumerate by
target ID, incl. SeekHead‚ÜíSeekHead/Tags/Chapters), and every `CueClusterPosition`
(+delta). Preserve all position VINT widths or FAIL-SAFE. Journaled temp +
best-effort recoverable swap; original untouched until the temp re-probes clean.

## 5. Matching (Phase B ‚Äî full detail in the design doc)

- Normalize languages: ISO 639-1 ‚Üí 639-2/T + the B/T doublets (`ger`‚Üí`deu`,
¬Ý `fre`‚Üí`fra`, ‚Ä¶). `jp` is invalid ‚Üí map to `jpn` defensively but flag it.
¬Ý `zh*` ‚Üí `zho` and DOCUMENT that Cantonese/Mandarin are not distinguished.
- **Absent language ‚Üí `und`, treated as UNKNOWN, never English (fail-closed).**
- **Region subtags are the #1 wrong-track risk:** if the SEED track carries a
¬Ý region (`pt-BR`, `es-419`, `en-GB`, `fr-CA`), REQUIRE a region match and leave
¬Ý region-mismatched files **unchecked**; only collapse region when the seed is
¬Ý region-less.
- Predicate: `hasAudio(audioLang) AND (subLang==NONE OR hasSub(subLang))`. Forced
¬Ý is NOT part of the predicate.
- Tie-break ladder (deterministic): already-default ‚Üí forced-alignment (subs
¬Ý only) ‚Üí de-prioritize commentary/SDH by title (soft, never eliminates the sole
¬Ý candidate) ‚Üí first in document order. If the resolved candidate is
¬Ý commentary-flagged AND is the SOLE candidate, surface **PARTIAL/needs-review**
¬Ý ‚Äî never silently default a commentary track.
- State plainly in the UI: **apply-to-similar matches BY LANGUAGE ONLY and does
¬Ý not replicate the seed's positional/title choice; the user must verify.**
